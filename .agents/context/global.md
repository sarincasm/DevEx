# Agent Context

## Obsidian
Use the `obsidian` skill when reading or writing vault notes, managing tasks, or fetching context from the KB folder.

Quick reference (without the skill):
- List vaults: `obsidian vaults`
- Read file: `obsidian read path="<path>" vault=<vault>`
- List folder: `obsidian files folder=<folder> vault=<vault>`
- Search: `obsidian search query="<query>" vault=<vault>`
